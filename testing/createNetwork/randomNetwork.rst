

net.randomNetwork(size, prop node=dynNode, edge=weightedEdge)
        Adds a directed random network

Parameters
----------
size : int
        Number of nodes to be added.
size : double \in [0,1]
        Propability for a connection between two nodes.
node : nodeTemplate
        Every added node is a copy of this template.
edge : edgeTemplate
        Every added edge is a copy of this template.
